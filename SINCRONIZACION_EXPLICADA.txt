TABLAS INVOLUCRADAS:
═════════════════════════════════════════════════════════════════════════════

web_orders (ORIGEN - tabla web)
├─ id (UUID)
├─ source = 'website'
├─ business_name
├─ contact_name
├─ phone
├─ items (JSON array)
└─ created_at

customers (DESTINO - tabla dashboard)
├─ id (integer, AUTO)
├─ whatsapp_phone
├─ name = business_name
├─ phone
├─ address = delivery_address
├─ status = 'active'
└─ user_uid = 'system'

orders (DESTINO - tabla dashboard)
├─ id (integer, AUTO)
├─ customer_id (FK customers.id)
├─ status = 'pending'
├─ total_amount = '0.00'
├─ user_uid = 'system'
├─ notes
├─ delivery_address
└─ created_at

order_items (DESTINO - tabla dashboard)
├─ id (integer, AUTO)
├─ order_id (FK orders.id)
├─ product_id
├─ product_name
├─ quantity
├─ status = 'PENDIENTE'
└─ created_at
